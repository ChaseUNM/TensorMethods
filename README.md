TensorMethods.jl is meant to aggregate evolution methods for tensor decompositions using ITensors. Currently the methods are the Rank-adaptive BUG method for Tucker-tensors and TDVP 1/2 for tensor trains. 

# TensorMethods.jl

This package is under active development and is not yet registered in the Julia General Registry.

---

## Installation (Unregistered Package)

Since this package is not registered, you must install it using `Pkg.develop`.

### Option 1: Install package after cloning repository

In a Julia REPL:

```julia
using Pkg
Pkg.develop(url="/path/to/TensorMethods")
```

### Option 2: Install package using git url

In a Julia REPL: 

```julia
using Pkg
Pkg.develop(url="https://github.com/ChaseUNM/TensorMethods")
```
